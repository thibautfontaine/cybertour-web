# Cyber Tour Reunion 2026

Site web officiel du Cyber Tour Reunion 2026, evenement cybersecurite organise par le CLUSIR Reunion Ocean Indien dans le cadre du Cybermois.

**22 Octobre 2026 — IUT / ESIROI, La Reunion**

## Stack

Astro 6 + Tailwind CSS 4, deploye sur GitHub Pages.

## Dev

```bash
npm install
npm run dev
```

## Deploy

Push sur `main` → GitHub Actions build + deploy sur GitHub Pages.

Domaine : `cybertour.re`
