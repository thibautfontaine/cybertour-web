# Cyber Tour Reunion 2026

Site web officiel du Cyber Tour Reunion 2026, evenement cybersecurite organise par le CLUSIR Reunion Ocean Indien dans le cadre du Cybermois.

**22 Octobre 2026 — IUT / ESIROI, La Reunion**

## Stack

Astro 6 + Tailwind CSS 4, deploye sur GitHub Pages.

## Dev

```bash
npm install
npm run dev
```

## Deploy

Push sur `main` → GitHub Actions build + deploy sur GitHub Pages.

Domaine : `cybertour.re`

## Contribuer

### Collaborateur 

1. Clone le repo (`git clone git@github.com:thibautfontaine/cybertour-web.git`)
2. Cree une branche depuis `main` (`git checkout -b feat/ma-feature`)
3. Fais tes modifications
4. Commit et push sur ta branche
5. Ouvre une Pull Request vers `main`

### Contributeur externe

1. Forke le repo
2. Clone ton fork
3. Cree une branche depuis `main` (`git checkout -b feat/ma-feature`)
4. Fais tes modifications
5. Commit et push sur ton fork
6. Ouvre une Pull Request vers `main` du repo principal

Seul le mainteneur peut approuver et merger les PRs.
